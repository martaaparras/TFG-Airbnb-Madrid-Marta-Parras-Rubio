# TFG-Airbnb-Madrid-Marta-Parras-Rubio
Análisis de precios en Airbnb Madrid

## Descripción

Este repositorio contiene el desarrollo del Trabajo Fin de Grado centrado en el análisis y predicción de precios de alojamientos Airbnb en la ciudad de Madrid mediante técnicas de Ingeniería del Dato, Analítica Predictiva y Machine Learning.

## Objetivos

- Analizar los factores que influyen en el precio de los alojamientos Airbnb.
- Realizar procesos de limpieza, transformación y enriquecimiento de datos.
- Construir modelos predictivos para la estimación de precios.
- Comparar diferentes algoritmos de Machine Learning.
- Extraer conclusiones útiles para la toma de decisiones.

## Tecnologías utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Google Colab

## Dataset

Los datos utilizados proceden de Inside Airbnb y fueron enriquecidos con información socioeconómica del Ayuntamiento de Madrid.
